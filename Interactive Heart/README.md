# Interactive Heart

Un corazón interactivo construido con HTML5 Canvas y JavaScript. Las partículas forman una figura de corazón que reacciona al movimiento del cursor y regresan suavemente a su posición original.

## ✨ Demo

Mueve el cursor cerca del corazón para dispersar las partículas y observa cómo vuelven a reconstruir la figura de forma fluida.

## 🚀 Tecnologías

* HTML5
* CSS3
* JavaScript
* Canvas API

## 🎨 Características

* Corazón generado mediante una ecuación matemática.
* Sistema de partículas interactivo.
* Reacción en tiempo real al movimiento del cursor.
* Animación suave con efecto de retorno.
* Renderizado utilizando Canvas API.

## ⚙️ Instalación

1. Clona el repositorio.
2. Navega hasta la carpeta del proyecto.
3. Abre `index.html` en tu navegador.

## 📚 Conceptos Practicados

* Canvas API
* Sistemas de partículas
* Animación con requestAnimationFrame

## 🎯 Objetivo del Proyecto

Practicar la creación de visualizaciones interactivas mediante Canvas, explorando sistemas de partículas, animaciones fluidas y representación matemática de formas complejas.
